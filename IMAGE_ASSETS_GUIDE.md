# Kumanyini Blocks Image Asset Guide

This document defines all images needed for the website, including:

- what each image should show
- recommended dimensions and formats
- where each file should live
- how to search and source suitable images

---

## 1) Folder Structure and Naming

Store images in:

- `assets/images/products/`
- `assets/images/gallery/`
- `assets/images/site/` (create this folder for hero, about, and general brand images)

Use lowercase, hyphenated names:

- `hero-block-yard.jpg`
- `product-5-inch-hollow.jpg`
- `gallery-delivery-truck-01.jpg`

Prefer `.jpg` for photos, `.png` only when transparency is needed, `.svg` for logos/icons.

---

## 2) Global Image Standards

- **Quality:** sharp, well-lit, realistic industrial scenes
- **Style:** professional, clean, concrete manufacturing / construction context
- **Color tone:** neutral-to-cool, should match sea-blue brand palette
- **Orientation:** use landscape unless stated otherwise
- **Compression target:** 150 KB to 500 KB per image where possible
- **Accessibility:** always add meaningful `alt` text

Recommended export sizes:

- Hero/background wide images: `1920x1080`
- Standard section images: `1200x800`
- Product card images: `1000x750` (4:3)
- Gallery images: `1200x800`
- Optional square thumbnails: `800x800`

## 4) Where to Search

Use reputable stock/photo sites with clear licenses:

- Unsplash
- Pexels
- Pixabay
- Wikimedia Commons (check license carefully)

For commercial usage, always verify:

- allowed commercial use
- attribution requirements
- no trademark/logo conflicts in photo

---

## 5) Search Strategy (Fast Workflow)

1. Start broad with core material + context:
  - `concrete blocks yard`
  - `block factory`
  - `paving blocks`
2. Add quality filters:
  - landscape orientation
  - high resolution
  - neutral/cool tone
3. Add regional relevance when possible:
  - `africa`, `ghana`, `tropical construction`
4. Shortlist 2-3 variants per slot.
5. Pick images with similar lighting/color so pages feel cohesive.

---

## 6) Final Pre-Upload Checklist

- Correct folder and filename used
- Image dimensions match recommended target
- File compressed (reasonable size)
- No visible watermark/logos
- Alt text written for each image
- Looks good on mobile and desktop crop
